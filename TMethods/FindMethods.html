Definition and Usage The find() method returns descendant elements of the selected element. A descendant is a child, grandchild, great-grandchild, and so on. The DOM tree: This method traverse downwards along descendants of DOM elements, all the way down to the last descendant. To only traverse a single level down the DOM tree (to return direct children), use the children() method. Note: The filter parameter is required for the find() method, unlike the rest of the tree traversal methods. Tip: To return all of the descendant elements, use the "*" selector.
